# SpotifyHelper
Helps to get duplicates from the Spotify playlist.

Access token can be requested here: https://developer.spotify.com/console/get-playlist/: 
 1) Click GetToken.
 2) Select scopes with playlist at name and press "Request token".

To get playlist Id select playlist options -> Share -> Copy Spotify URI.
E.g, link to the playlist is https://open.spotify.com/playlist/20i6gDaOmjsFhYRuYAwsRA?si=bc395093a8264f18
, so the PlaylistId is 20i6gDaOmjsFhYRuYAwsRA.
